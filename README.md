# vue-excharge

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

## Para asegurarse de que no tenga errores el codigo
```
npm run lint
```

## Para empezar a correr el servidor en el navegador.
```
serve -s dist
```
